SPSSINC MFP GLM Extension Command

Estimate a regression with fractional polynomials for the scale variables

SPSSINC MFP GLM DEPENDENT=dependent variable*
COVAR = scale independent variables
FACTOR = categorical independent variables
VARIABLEALPHA = significance or list of significances, one for each variable
FPALPHA = significance or list of significances
DF = degrees of freedom or list of dfs, one for each scale variable
FAMILY = GAUSSIAN** or BINOMIAL or POISSON or GAMMA or INVERSEGAUSSIAN or QUASI
LINK = link name

/OPTIONS MISSING=LISTWISE** or FAIL
PLOTS = YES or NO**

/SAVE PROGRAMFILE = “filespec”
RESIDUALSDATASET = dataset
COEFSDATASET = dataset.

/HELP

* Required
** Default

SPSSINC MFP GLM /HELP prints this information and does nothing else.

SPSSINC MFPGLM DEPENDENT=mpg COVAR=engine weight FACTOR=origin
DF=3 3.

Split files and weight are not honored by this command.

DEPENDENT specifies the dependent variable. Continuous predictors are specified as COVAR and factors as FACTOR. Categorical independent variables are automatically converted appropriately to factors. A constant term is automatically included.

VARIABLEALPHA specifies the significance level for inclusion of variables, and FPALPHA specifies the significance level for polynomial terms. Both can be a single number applied to all relevant variables or a list of numbers. If a list, there must be as many values as the total number of predictors (VARIABLEALPHA) or the number of covariates. Use 1 for non-polynomial covariates.

DF, also a single number or a list of numbers, specifies the parameters for the polynomials. The choices are

  • 1: linear model
  • 2: FP model with maximum degree 1,
  • 4: FP model with maximum degree 2.

FAMILY specifies the model.

LINK specifies the link function. Valid values depend on the family. mu stands for 1/mu^2 The choices are

  • GAUSSIAN: IDENTITY** or LOG or INVERSE
  • BINOMIAL: LOGIT** or PROBIT or CAUCHIT or CLOGLOG
  • GAMMA: INVERSE** or IDENTITY or LOG
  • POISSON: log** or IDENTITY or SQRT
  • INVERSEGAUSSIAN: MU or INVERSE or IDENTITY or LOG
  • QUASI: LOGIT** or PROBIT or CLOGLOG or IDENTITY or INVERSE or LOG or MU or SQRT

OPTIONS

MISSING = LISTWISE causes listwise deletion of missing values. FAIL stops the procedure if missing values are encountered.

PLOTS determines whether plots are produced.

SAVE

PROGRAMFILE causes the R code for this procedure to be written to the specified file. The generated program can be a useful starting point for additional specifications.

RESIDUALSDATASET causes a dataset containing the residuals to be created. The dataset name must not already be in use. The case number is included as cases will only be written for input cases with no missing data. Data filtered by SPSS Statistics are not passed to R and will not generate cases in the residuals dataset.

COEFSDATASET causes a new dataset containing the coefficients to be created. The dataset name must not already be in use.
